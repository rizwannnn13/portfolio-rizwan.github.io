Install Package

Untuk menginstall package apache2, gunakan command berikut

apt install apache2

Ini adalah tampilan default web kita setelah menginstall apache2

Setelah itu masuk ke dalam folder apache2

cd /var/www/html

Lalu hapus semua file yang ada di folder tersebut

rm * /var/www/html

Konfigurasi

Buat file index.html baru

nano index.html

Disini saya menggunakan file html yang sudah ada dan memindahkannya memakai FileZilla

Ubah resolv.conf menjadi nama domain dan IP DNS kalian masing-masing

nano /etc/resolv.conf


nameserver 10.1.31.1
search rizwanpemula.com

Restart apache2 kalian

/etc/init.d/apache2 restart
