Tambahkan User

Menambahkan user baru untuk diberikan akses ke service FTP

adduser (userkalian)

Contohnya saya akan pakai command adduser rizwanftp lalu berikan password sesuai selera kalian.

Install Package

Lalu install package untuk menjalankan service FTP yaitu proftpd

apt install proftpd

Masuk ke file yang telah kita install tadi

nano /etc/proftpd/proftpd.conf

Konfigurasi

Setelah itu kalian edit file berupa hapus tanda pagar "#" dan tambahkan user baru yang telah kalian buat.

Buat directory untuk digunakan sebagai FTP Server

mkdir /home/(user)/ftp

Sebagai contoh saya akan buat directory seperti mkdir /home/rizwanftp/ftp


Lalu setelah itu ubah hak akses untuk directory ftp

chmod 777 /home/(user)/ftp

Contoh: chmod 777 /home/rizwanftp/ftp .Keterangan: chmod 777 memberikan "SEMUA" permission ke semua jenis user (owner, group dan lain-lain).

Setelah itu restart service FTP Server

/etc/init.d/proftpd restart
