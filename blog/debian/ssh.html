Install Package SSH

Install SSH dengan menggunakan command

apt install ssh

Lalu masuk pada menu ssh

nano /etc/ssh/sshd_config

Konfigurasi Package

Edit file ssh tersebut dengan menghapus "#" pada bagian "PermitRootLogin" dan setel nilainya menjadi "yes"

#LoginGraceTime 2m
PermitRootLogin yes
#StrictModes yes

Lalu restart package ssh

/etc/init.d/ssh restart
