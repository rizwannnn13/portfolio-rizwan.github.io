Install Package

Package yang dibutuhkan adalah

  • Php
  • MySql
  • Mariadb
  • Unzip
apt install php7.3 php7.3-mysql mariadb-server unzip

Konfigurasi Database

setelah semua paket yang dibutuhkan telah terinstall, berikutnya adalah melakukan konfigurasi keamanan mysql dan database serta user yang menggunakan database nantinya. caranya adalah :

mysql_secure_installation

Ini berfungsi untuk mengatur kembali mysql yang telah terinstall dan membuat password baru untuk kemanan database pada mysql.

Selanjutnya kita membuat database dan user di mysql. Untuk masuk ke menu sql ketikan

mysql -u root -p

Contoh : saya akan membuat database dengan nama "db_pemula" dan user "rizwanpemula" dengan password "123". Maka ini command yang harus dijalankan.

CREATE DATABASE db_pemula;

CREATE USER 'rizwanpemula'@'localhost' IDENTIFIED BY '123';

GRANT ALL PRIVILEGES ON *.* TO 'rizwanpemula'@'localhost';

FLUSH PRIVILEGES;

EXIT;

Install Wordpress

Buatlah direktory untuk tempat kita menyimpan file wordpress

mkdir /var/www/wordpress

mkdir /var/www/wordpress/log

Setelah itu Ubah owner dan permission DocumentRoot wordpress

chmod 777 /var/www/wordpress

lalu pindah ke folder wordpress dengan perintah cd /var/www/wordpress untuk menginstall wordpress. Untuk penginstallan wordpress bisa menggunakan command ini.

wget https://wordpress.org/latest.zip

setelah itu kita ekstarak zip wordpress yang telah kita install

unzip latest.zip

lalu masuk ke dalam folder wordpress yang telah kita ekstrak tadi cd /wordpress lalu pindahkan semua file yang ada di dalam folder tersebut

mv * ../

Virtual Host

Tahap berikutnya kita akan membuat file VirtualHost untuk php. masuk terlebih dahulu ke /var/www/ lalu kita akan membuat file untuk php

mkdir /php

mkdir /php/log

Masuk ke folder php cd /php lalu kita buat file untuk VirtualHost. nano index.php

lalu masuk kedalam folder apache2 untuk menambahkan file VirtualHost untuk php dan wordpress. cd /etc/apache2/sites-available/

nano php.conf

Selanjutnya kita akan membuat file VirtualHost untuk wordpress

nano wordpress.conf

Setelah itu aktifkan file VirtualHost yang tadi kita telah buat.

a2ensite wordpress.conf

a2ensite php.conf

systemctl reload apache2

Setup Wordpress

Buka aplikasi web lalu buka subdomain untuk wordpress. Disini saya menggunakan subdomain wordpress.rizwanpemula.com

Gunakan database dan user yang telah kita buat di mysql tadi.

Konfigurasi Selesai

Selamat kita telah selesai konfigurasi wordpress di Debian 10. Buka wp admin menuju dashboard wordpress dengan user yang telah kita buat tadi, dan keluar dari menu wp adminn untuk melihat progres website wordpress kita.
